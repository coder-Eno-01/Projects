## About

This repository contains a range of projects I’ve built to strengthen my programming skills and explore different concepts in web development.

Each project focuses on practical implementation, from game logic and UI design to working with APIs and backend services.

  A weather application that fetches real-time data from external APIs and displays it in a clean interface.

- **30 Seconds Game (Coming Soon)**
